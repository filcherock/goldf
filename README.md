Goldf is a file manager written in Python. This manager is presented in two variations, in GUI and CLI

# Download
```
git clone https://github.com/filcherock/goldf.git
cd goldf

# CLI version
python src/terminal/goldf.py

# GUI version
python src/gui/goldf.py
```
